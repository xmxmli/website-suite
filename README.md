# Website Suite

Monorepo for an email sending API website and a future dashboard.

## Stack

- Turborepo
- pnpm workspace
- Next.js
- TypeScript
- Tailwind CSS

## Apps

- `apps/website`: public website placeholder.
- `apps/dashboard`: dashboard placeholder.

## Packages

- `packages/ui`: shared React UI components.
- `packages/config`: shared TypeScript and ESLint config files.

## Commands

```bash
pnpm install
pnpm dev:website
pnpm dev:dashboard
pnpm dev
pnpm build
pnpm lint
pnpm typecheck
```
